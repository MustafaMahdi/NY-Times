### `Clone the repository`:

### `git clone https://github.com/MustafaMahdi/NY-Times.git`

### `Navigate to the project directory:`:

### `cd NY-Times`

### `Install dependencies`:

### `npm install`

### `Start the development server`:

### `npm start`

### `Run tests`:

### `npm test`

### `Build the project`:

### `npm run build`
